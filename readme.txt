== Frequently Asked Questions ==

= I read that this plugin is no longer maintained. What should I do if I still use it on my website? =
First of all, don't panic :)
You can simply switch to the new [GetLoy payment gateway for WooCommerce](https://wordpress.org/plugins/wc-getloy-gateway/) plugin, which supports PayWay by ABA Bank and Pi Pay.
Once you set up the new plugin, you can safely uninstall PayWay for WooCommerce.
